# Changelog

All notable changes to the "Dark Moonlight" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.4] - 2024-12-17

### Added
- Extension icon (crescent moon with moonlight glow)

## [1.0.3] - 2024-12-17

### Added
- .vscodeignore file for cleaner extension packaging

## [1.0.2] - 2024-12-17

### Fixed
- Repository, bugs, and homepage URLs now use correct GitHub username

## [1.0.1] - 2024-12-17

### Changed
- Updated publisher name

## [1.0.0] - 2024-12-17

### Added
- Initial release of Dark Moonlight theme
- Complete UI color definitions based on Dark Modern
- Syntax highlighting with moonlight accents (#e0e6f0)
- Semantic token colors for enhanced language support
- Terminal ANSI colors with moonlight blue replacement
- Full coverage of VS Code workbench colors:
  - Activity bar
  - Side bar
  - Editor and editor groups
  - Tabs
  - Status bar
  - Panel and terminal
  - Notifications
  - Git decorations
  - Diff editor
  - Peek view
  - Debug toolbar
  - Settings UI
  - And more...

### Changed
- All blue accent colors replaced with moonlight gray-white (#e0e6f0):
  - `#007acc` → `#e0e6f0`
  - `#0078d4` → `#e0e6f0`
  - `#0e639c` → `#e0e6f0`
  - `#4fc1ff` → `#e0e6f0`
  - `#3794ff` → `#e0e6f0`
  - `#569cd6` → `#e0e6f0`
